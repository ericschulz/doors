Doors-Experiment 1

Experiment 1: 1 animal vs. 8 animals

This is the analysis for our first doors experiment, where children show increased motivation, i.e. search longer, if they can gain more information. In this first experiment, 49 children were asked to search for an animal by opening doors. Crucially, the animal never appears but we manipulated the number of possible animals the final animal could be, i.e. 1 vs. 8. The prediction is that finding out which of 8 animals it would be is more informative than knowing the animal from the start. Thus, children should search longer in the 8 animals condition.

Finally, let’s do some plotting. We begin by setting up some basic plotting functions.

#standard errors
se<-function(x){sd(x)/sqrt(length(x))}

#data summary function
data_summary <- function(x) {
  m <- mean(x, na.rm=TRUE)
  sem <-sd(x, na.rm=TRUE)/sqrt(sum(!is.na(x)))
  ymin<-m-1.96*sem
  ymax<-m+1.96*sem
  return(c(y=m,ymin=ymin,ymax=ymax))
}

#data summary function
data_summary <- function(x) {
  m <- mean(x, na.rm=TRUE)
  sem <-sd(x, na.rm=TRUE)/sqrt(sum(!is.na(x)))
  ymin<-m-1.96*sem
  ymax<-m+1.96*sem
  return(c(y=m,ymin=ymin,ymax=ymax))
}

#binomial function
mybinomfunc<-function(x, mark){
  b1<-binom.test(sum(x==mark), length(x), p=1/2, conf.level=0.95)
  dout<-data.frame(state=mark, p=as.numeric(b1$estimate), clow=b1$conf.int[1], cup=b1$conf.int[2])
  return(dout)
}
